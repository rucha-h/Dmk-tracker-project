The General structure of the project.
Dashboard : Has overview of of the current status.
Characters: List of all characters. 
            Welcome the characters & track their levels.
            Wishlist characters not yet welcome & which you want next.
Attraction: List of all Attractions.
            Track which ones you have built.
Costumes: List of all Costumes.
          Track which ones you own.
Tokens: Track the tokens of characters you have welcomed or wishlisted.
        It shows which tokens are needed in what quantity based on your data from characters.
        Each token has dropdown which shows where to farm the tokens.
Floats: List of all Floats.
        Track which ones you own & active.
Consessions: List of all consessions.
            Track which ones you own.
Enchantments: List of all enchantments available with levels.
Campaign: Track the Main Story Line Act 1-6.

Data is Based on before update 103.
I will try to update the data after events when I get time.
