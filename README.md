# 🏰 Disney Magic Kingdoms Tracker

A fan-made companion tool to help you track your progress in Disney Magic Kingdoms — characters, attractions, costumes, tokens, floats, concessions, enchantments, and more — all in one place, right in your browser. No account needed, no installation required.

> ⚠️ Data is based on the game state **till Update 104**. Updates will be added over time.

## 📥 How to Download & Open

You don't need to be tech-savvy to use this! Just follow these steps:

### Step 1 — Download the project

1. Go to 👉 [https://github.com/rucha-h/Dmk-tracker-project](https://github.com/rucha-h/Dmk-tracker-project)
2. Click the green **`< > Code`** button near the top right
3. Select **"Download ZIP"** from the dropdown

### Step 2 — Unzip the folder

- **Windows:** Right-click the downloaded `.zip` file → click **"Extract All"**
- **Mac:** Double-click the `.zip` file — it extracts automatically

### Step 3 — Open the tracker

1. Open the extracted folder
2. Find the file called **`index.html`**
3. Double-click it — it will open in your web browser

✅ That's it! No installation, no sign-in, no internet needed after download.

## ✨ Features

Here's what you can track with this tool:

| Section | What It Does |
|---|---|
| 🏠 **Dashboard** | A quick overview of your current game progress at a glance |
| 🧑‍🤝‍🧑 **Characters** | See all characters, mark which ones you've welcomed, and set up a wishlist for who you want next |
| 🎢 **Attractions** | Browse all attractions and mark which ones you've built |
| 👗 **Costumes** | Full costume list — track which ones you own |
| 🪙 **Tokens** | See exactly which tokens you need and how many, based on your characters. Each token has a dropdown showing where to farm it |
| 🎡 **Floats** | Track which floats you own and which are currently active |
| 🍦 **Concessions** | Browse all concessions and mark which ones you own |
| ✨ **Enchantments** | View all available enchantments and their levels |
| 📖 **Campaign** | Track your progress through the Main Story Line (Acts 1–6) |

## ❓ Frequently Asked Questions

**Do I need an internet connection to use this?**
Only for the initial download. Once you have the files, everything works offline.

**Does this connect to my Disney Magic Kingdoms game account?**
No — this is a standalone tracker. You enter your progress manually. Your data stays on your device.

**Where is my data saved?**
Your data is saved in your browser's local storage, so it stays on your device. Clearing your browser data may reset it.

**Will it work on my phone or tablet?**
It's designed to open in a browser, so it should work on most devices. For the best experience, a desktop or laptop is recommended.

**The data seems outdated — what's going on?**
The data reflects the game before **Update 103**. The project is actively maintained and updates will be added over time.

**Something looks broken. What should I do?**
Make sure you extracted the ZIP fully before opening `index.html`. Opening it directly from inside the ZIP won't work correctly.
